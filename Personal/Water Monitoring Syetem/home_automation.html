SYSTEM ARCHITECTURE

Hardware & Software Layers

Distributed architecture utilizing ESP32 for complex IO-heavy rooms and ESP8266 for smaller footprints, orchestrated by Home Assistant.

🏢

Zone A: Main Room & Lab

ESP32 DEVKIT V1

High-performance nodes handling complex automation tasks with multiple I/O requirements.

  • Multiple relay outputs for lighting control
  • Electromagnetic door locks with fail-safe
  • Magnetic reed switches for door status
  • Real-time state synchronization
🏠

Zone B: HR Department

NODEMCU ESP8266

Compact nodes optimized for lighting and presence detection in smaller spaces.

  • Lighting control with dimming support
  • PIR motion sensing (HC-SR501)
  • Presence-based automation triggers
  • Energy-efficient operation
⚡

Relay Modules

Lab Room Configuration Excerpt
# Lab Room - ESP32 Home Automation Controller
esphome:
  name: test
  friendly_name: Lab Room

esp32:
  board: esp32dev
  framework:
    type: arduino

# Network Stability with Static IP
wifi:
  manual_ip:
    static_ip: 192.168.1.150
    gateway: 192.168.1.1
    subnet: 255.255.255.0

# Hybrid Switching: Physical + Digital Control
binary_sensor:
  - platform: gpio
    id: wall_switch_light_1
    pin:
      number: GPIO25
      mode: INPUT_PULLUP
      inverted: true
    on_state:
      then:
        - if:
            condition:
              binary_sensor.is_on: wall_switch_light_1
            then:
              - light.turn_on: room_light_1
            else:
              - light.turn_off: room_light_1

  - platform: gpio
    pin: GPIO13
    name: "PIR Motion Sensor"
    device_class: motion

# Relay Outputs (Active-Low for Safety)
light:
  - platform: binary
    id: room_light_1
    name: "Room Light 1"
    output: relay_light_1
    restore_mode: RESTORE_DEFAULT_ON

switch:
  - platform: output
    name: "Door Lock"
    output: relay_door_lock
    restore_mode: ALWAYS_OFF
PORTFOLIO HIGHLIGHTS

Key Achievements

Technical innovations that set this automation system apart from consumer solutions.

🔄

Hybrid Switching

Successfully solved the "smart switch" dilemma by integrating physical toggles that update the digital state instantly without breaking the circuit.

⚡

Energy Optimization

Implemented a 5-minute inactivity timeout across all zones, reducing unnecessary electricity consumption in unoccupied rooms.

🔒

Security Integration

Centralized door monitoring and locking, providing a single source of truth for facility security within the Home Assistant dashboard.
