# PCB

The Borgo Pumps Rev A controller currently uses an Arduino Nano, SIM800L module, relay module, pair button, and external power wiring. No fabrication-ready PCB project has been committed yet.

When a PCB is designed, place the KiCad or EasyEDA source here under its hardware revision. Do not place Fusion 360 casing source files in this repository. Attach finished casing exports only to a tagged GitHub Release and the Borgo Labs Google Drive release folder.
