§ 01 Project Overview
This document records the design, research, and planned improvements for the CNC machine asset at Southern IoT. It is intended for internal archive and technical reference.
Objective

Transform a non-operational CNC router into a fully reliable, production-ready machine that can be used for routing custom mechanical parts — including components for other in-house projects such as the wire-bending machine. The longer-term goal is to integrate the machine into a single-table multi-process fabrication station.

Current Use
  • Routing custom mechanical parts for internal project builds.
  • Primary consumer at present: Wire Bending Machine component fabrication.
  • Occasional engraving and profiling of acrylic and wood workpieces.

§ 03 CNC Table Design
Concept

A dedicated fabrication table was designed to house both a CNC router and a Laser-CNC unit side by side on a single rigid frame. This allows simultaneous or sequential multi-process fabrication without relocating workpieces — improving throughput and reducing setup time.

Key Design Goals
  • Shared structural frame — eliminates two separate setups
  • Defined zones: router zone left, laser zone right
  • Common electronics bay for PSU and controllers
  • Cable management channels integrated into frame
  • Adjustable leveling feet for floor compensation
Benefits
  • One table → two processes in parallel
  • Reduced footprint vs. two separate machines
  • Single power distribution point
  • Professional, clean production environment

§ 04 Planned Improvements

The following upgrades are planned once required components are sourced. These will elevate the machine from "working" to "professional one-tap operation."

01
Limit Switch Installation — Install 3-axis homing switches (X, Y, Z) and enable GRBL homing cycle ($22=1). This removes the need for manual origin-setting and enables automatic soft limits for axis protection.
02
Professional Cable Management — Route all stepper, spindle, and limit switch wiring through drag chains. Label all connectors. Eliminate exposed cabling that currently poses a snag/short risk.
03
Enclosure for Electronics — Mount controller board, stepper drivers, PSU, and emergency stop into a dedicated enclosure panel beneath or beside the machine. Adds protection and gives a finished product appearance.
04
One-Tap Operation Workflow — With homing switches in place, the target workflow is: power on → auto-home → load job file → press Start. No manual G92 offset procedure required.
05
CNC Table Build — Fabricate the designed dual-process table. Install both the router and laser-CNC on the same frame. Commission both machines from the shared electronics bay.
